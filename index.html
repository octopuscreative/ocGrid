Octopus Grid

Essentially, the Octopus Grid is a 12 column grid that has adjustable behaviors for its "collapsed" and "expanded" states. Any given "row" of the grid will have:

  • a "collapsed fluid" state for <640
  • a "squished normal" state for 640 ---> <960
  • a "normal" state for 960 ---> <1280
  • an "expanded state" for >= 1280

For example, if you used the class .row.clp8.exp16, it would be 12 columns for tablets and "normal" desktop, but on mobile it would be 8 columns and on "large" desktop it would be 16 columns. Using this setup, mobile would fit two 4-column items, "large" desktop would fit four 4-column items, and everything else would fit 3 as per "normal"
